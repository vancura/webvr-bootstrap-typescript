# TypeScript WebVR Bootstrap

This is a simple WebVR/Three.js bootstrap on top of webvr-polyfill in TypeScript, with BrowserSync, powered by Gulp.

Some other features:

* normalize-css
* CSS autoprefixer
* TSLint
* sourcemaps


## Author

Vaclav Vancura ([github.com/vancura](https://github.com/vancura))


## Installation

```
npm install
bower install
gulp
```
